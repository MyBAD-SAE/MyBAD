Guide d'utilisation
MyBAD

Application de gestion de tournois de badminton en milieu scolaire. Ce guide vous accompagne de la création de votre compte jusqu'à la clôture d'une séance.

🏸

Joueurs (élèves)

Interface mobile — déclarez vos matchs, suivez votre ELO et votre classement.

🎓

Administrateurs (profs)

Interface desktop — gérez vos cours, vos séances et les règles ELO.

🏸

Guide Joueur

Application mobile — à utiliser sur smartphone uniquement

1. Introduction

MyBAD est une application qui permet de gérer les tournois de badminton de votre cours. Elle calcule automatiquement vos points ELO après chaque match, tient à jour un classement en temps réel et conserve l'historique de toutes vos parties.

💡

Application mobile uniquement. MyBAD est conçue pour être utilisée sur smartphone. Ouvrez-la dans le navigateur de votre téléphone (Chrome, Safari…) à l'adresse fournie par votre professeur.

Qu'est-ce que l'ELO ?

L'ELO est un système de points utilisé pour évaluer le niveau des joueurs. Chaque victoire vous rapporte des points, chaque défaite vous en retire. Plus vous gagnez avec une différence de score élevée, plus la victoire est récompensée. Votre classement évolue en permanence au fil des matchs.

2. Premiers pas

Créer un compte

Rendez-vous sur la page de connexion de MyBAD. Sélectionnez l'onglet Inscription et renseignez :

  • Prénom et Nom
  • Adresse email
  • Mot de passe (8 caractères minimum)

Vous pouvez aussi vous inscrire directement avec votre compte Google en appuyant sur le bouton dédié.

Se connecter

  1. Ouvrez MyBAD sur votre smartphone.

  2. Saisissez votre email et votre mot de passe, puis appuyez sur Se connecter.

  3. Ou appuyez sur Continuer avec Google si vous vous êtes inscrit avec Google.

Mot de passe oublié

Sur la page de connexion, appuyez sur Mot de passe oublié ?. Saisissez votre email : vous recevrez un lien de réinitialisation valable quelques heures.

Créer votre code PIN

Lors de votre première connexion, l'application vous demande de définir un code PIN à 4 chiffres. Ce code sert à valider les matchs que vos adversaires déclarent contre vous — vous devrez le saisir pour confirmer le résultat.

💡

Vous pourrez modifier votre PIN à tout moment depuis Mon profil → Informations personnelles.

En attente d'un cours

Après votre première connexion, si vous n'êtes inscrit dans aucun cours, l'application affiche un écran d'attente avec votre code joueur unique à 6 chiffres.

💡

Donnez votre code à votre professeur. C'est le seul moyen pour lui de vous ajouter à un cours. Vous pouvez appuyer sur le code pour le copier dans le presse-papier.

3. Tableau de bord

Une fois inscrit dans un cours, le tableau de bord devient votre page d'accueil. Il centralise toutes vos informations en un coup d'œil.

Carte ELO

Votre score ELO actuel est affiché en haut. La variation (ex : +12 ou -8) indique l'évolution de votre ELO sur vos 4 dernières séances.

Statistiques

  • Total de matchs joués dans ce cours
  • Victoires et Défaites
  • Taux de victoires en pourcentage

Widget classement

Les 8 premiers joueurs du cours sont affichés avec leur rang et leur ELO. Votre position dans le classement est mise en évidence.

Derniers matchs

Les matchs les plus récents apparaissent en bas du tableau de bord. Pour voir l'intégralité de l'historique, rendez-vous dans l'onglet Matchs.

Changer de cours

Si vous êtes inscrit dans plusieurs cours, un sélecteur apparaît en haut de l'écran. Appuyez dessus pour basculer d'un cours à l'autre.

4. Déclarer un match

⚠️

Prérequis : votre professeur doit avoir lancé une séance. Le bouton de déclaration est grisé si aucune séance n'est active.

Pour déclarer un match, appuyez sur le bouton + au centre de la barre de navigation. Le formulaire se déroule en 4 étapes.

Étape 1 — Choisir votre adversaire

La liste affiche tous les joueurs présents dans la séance active. Appuyez sur le joueur contre lequel vous venez de jouer.

Étape 2 — Saisir les scores

Entrez le score de chaque joueur. Au moins un des deux scores doit être supérieur ou égal à 15.

Étape 3 — Valider avec le PIN de l'adversaire

Pour confirmer le résultat, votre adversaire doit saisir son code PIN à 4 chiffres. Passez le téléphone à votre adversaire le temps qu'il entre son PIN. Appuyez sur Valider les résultats pour enregistrer définitivement le match.

Étape 4 — Résumé et confirmation

Un résumé affiche le match (joueurs, scores) et la variation ELO pour chacun.

Erreurs fréquentes

MessageCauseSolution
Aucune séance activeLe professeur n'a pas lancé de séanceAttendez que la séance soit ouverte
Déjà joué contre cet adversaireUn match contre ce joueur existe déjà dans la séanceChoisissez un autre adversaire
PIN incorrectMauvais code saisiL'adversaire re-saisit son PIN

5. Historique des matchs

Appuyez sur l'icône Matchs dans la barre de navigation pour consulter tous vos matchs joués.

Filtrer les matchs

  • Appuyez sur Tous pour voir l'ensemble des matchs
  • Appuyez sur Victoires pour n'afficher que les victoires
  • Appuyez sur Défaites pour n'afficher que les défaites

Rechercher un adversaire

Utilisez la barre de recherche en haut pour filtrer par nom d'adversaire.

Lire une ligne de match

  • Icône verte = victoire / icône rouge = défaite
  • Score : votre score – score de l'adversaire
  • Variation ELO : +12 en vert ou -8 en rouge
  • Date du match

6. Classement

Appuyez sur l'icône Trophée dans la barre de navigation pour accéder au classement général du cours.

7. Détails ELO

Accessible depuis le tableau de bord en appuyant sur votre carte ELO. Cette page vous donne une vue détaillée de l'évolution de votre score.

Graphique d'évolution

Le graphique montre l'évolution de votre ELO dans le temps. Utilisez les boutons 30j (30 derniers jours) ou Tout pour ajuster la période affichée. Appuyez sur un point du graphique pour afficher la valeur exacte à cette date.

Statistiques clés

  • ELO actuel : votre score du moment
  • Meilleur ELO : votre record historique dans ce cours
  • Classement actuel : votre rang parmi tous les joueurs

8. Mon profil

Appuyez sur l'icône Profil (dernière icône à droite de la barre de navigation) pour accéder à votre espace personnel.

Modifier la photo de profil

Appuyez sur l'icône caméra sur votre avatar pour sélectionner une nouvelle photo depuis votre galerie. Les formats JPG, PNG et WebP sont acceptés.

Informations personnelles

Dans Informations personnelles, vous pouvez modifier :

  • Nom et prénom
  • Adresse email
  • Mot de passe (un indicateur de force vous guide)

Appuyez sur Enregistrer pour valider les modifications.

Configurer son PIN

Le PIN est un code à 4 chiffres que vos adversaires doivent saisir pour valider un match que vous avez joué. Il est indispensable pour déclarer des matchs.

  1. Allez dans Informations personnelles.

  2. Dans la section Code PIN, saisissez votre PIN actuel puis le nouveau PIN.

  3. Confirmez le nouveau PIN et appuyez sur Enregistrer.

⚠️

Ne communiquez votre PIN à personne. Le PIN sert uniquement à valider les déclarations de matchs — il n'est pas lié à votre connexion.

Changer de cours actif

Si vous êtes inscrit dans plusieurs cours, vous pouvez choisir le cours affiché sur votre tableau de bord depuis l'option Cours actif dans votre profil.

Se déconnecter

Appuyez sur le bouton Déconnexion en bas de votre profil. Une confirmation vous sera demandée.

Télécharger mes données (RGPD)

MyBAD vous permet de télécharger l'ensemble des données vous concernant. Dans votre profil, appuyez sur Confidentialité pour accéder à cette option.

Supprimer mon compte

La suppression du compte est définitive et irréversible. Toutes vos données (matchs, ELO, profil) seront effacées. Pour procéder, rendez-vous dans votre profil et appuyez sur Supprimer mon compte.

⚠️

La suppression de votre compte supprime également votre historique de matchs et vos points ELO dans tous les cours auxquels vous participez. Cette action est irréversible.

Guide Administrateur

Interface desktop — à utiliser sur ordinateur uniquement

1. Introduction

L'espace administrateur est dédié aux professeurs et responsables de cours. Il permet de créer et gérer des cours, d'organiser des séances de jeu, de suivre les matchs et de configurer les règles ELO.

💡

Interface desktop uniquement. L'espace administrateur est optimisé pour les écrans d'ordinateur. L'accès depuis un smartphone affiche un message d'erreur.

Déroulement type d'une séance

  1. Créez un cours et ajoutez vos joueurs.

  2. Lancez une séance depuis le tableau de bord.

  3. Les élèves déclarent leurs matchs sur leur téléphone pendant la séance.

  4. Clôturez la séance en fin de cours.

  5. Consultez le classement mis à jour.

2. Premiers pas

Créer un compte administrateur

Rendez-vous à l'adresse /admin/login depuis un navigateur desktop. Sélectionnez l'onglet Inscription et renseignez vos informations (nom, prénom, email, mot de passe).

Se connecter

Saisissez votre email et mot de passe puis appuyez sur Se connecter. Vous arrivez directement sur le tableau de bord administrateur.

Navigation

La barre latérale gauche donne accès à toutes les sections :

  • Dashboard — vue d'ensemble du cours sélectionné
  • Joueurs — gérer les participants
  • Matchs — consulter et modifier les matchs
  • Règles et défis — configurer les points ELO
  • Administrateurs — gérer les co-admins (si vous êtes l'admin principal)

3. Créer un cours

Lors de votre première connexion, aucun cours n'existe. Cliquez sur Créer mon premier cours depuis le tableau de bord.

Remplir le formulaire

Cliquez sur Créer le cours. Le cours apparaît immédiatement dans le sélecteur de classe du tableau de bord.

💡

Vous pouvez créer autant de cours que nécessaire. Basculez entre vos cours avec le sélecteur en haut du tableau de bord.

Supprimer un cours

Dans le tableau de bord, faites défiler jusqu'à la section Zone de danger en bas de page. Cliquez sur Supprimer ce cours, lisez les avertissements, tapez SUPPRIMER pour confirmer.

⚠️

La suppression d'un cours efface définitivement tous les joueurs, matchs, séances et données ELO associés. Cette action est irréversible.

4. Gérer les joueurs

Accédez à la page Joueurs depuis la sidebar. Vous y retrouvez la liste complète des participants au cours sélectionné.

Ajouter un joueur

  1. Demandez à l'élève son code joueur à 6 chiffres (affiché sur son tableau de bord ou son profil).

  2. Cliquez sur Ajouter un joueur en haut à droite.

  3. Saisissez le code et l'ELO de départ (1 000 par défaut).

  4. Cliquez sur Ajouter.

💡

L'ELO de départ peut être ajusté pour les joueurs ayant déjà un niveau connu. Laissez 1 000 si vous partez sur une base équitable pour tous.

Modifier un joueur

Cliquez sur l'icône crayon à droite du joueur. Vous pouvez :

  • Ajuster manuellement l'ELO (une entrée sera créée dans l'historique)
  • Activer ou désactiver le joueur
  • Nommer co-administrateur : le joueur aura accès à l'espace admin pour ce cours

Retirer un joueur

Cliquez sur l'icône poubelle à droite du joueur, puis confirmez. Ses matchs et son historique ELO dans ce cours sont supprimés.

5. Lancer une séance

Une séance représente une session de jeu pendant laquelle les élèves peuvent déclarer leurs matchs. Sans séance active, les joueurs ne peuvent pas enregistrer de résultats.

Démarrer une séance

  1. Depuis le tableau de bord, cliquez sur la carte Nouvelle séance. Ou cliquez directement sur Séance dans la sidebar.

  2. Cliquez sur Lancer une nouvelle séance.

  3. La séance est immédiatement active — les joueurs peuvent commencer à déclarer leurs matchs.

Pendant la séance

La page Séance affiche en temps réel :

  • Le classement mis à jour (podium + grille)
  • Les derniers matchs déclarés (les 10 plus récents)
⚠️

La page Séance n'affiche que les données et statistiques de la séance en cours, pas globales.

Vous pouvez également ouvrir le classement en plein écran pour le projeter au tableau.

✅

Une icône verte sur la carte Séance du dashboard indique qu'une séance est en cours. Les joueurs voient le bouton + activé sur leur téléphone.

Clôturer la séance

  1. Cliquez sur Clôturer la séance en bas à droite de la page Séance.

  2. Confirmez dans la fenêtre de dialogue.

  3. Les ELO sont définitivement enregistrés. Les joueurs ne peuvent plus déclarer de matchs pour cette séance.

⚠️

Une fois clôturée, une séance ne peut pas être rouverte. Si un match a été oublié, vous pouvez toujours le corriger depuis la page Matchs (les ELO seront recalculés automatiquement).

6. Gérer les matchs

La page Matchs liste l'ensemble des matchs joués, regroupés par séance.

Naviguer entre les séances

Des onglets horizontaux en haut de la liste permettent de filtrer par séance. L'onglet Tout affiche tous les matchs de toutes les séances confondues.

Modifier un match

Cliquez sur l'icône crayon à droite du match. Ajustez les scores des deux joueurs et validez. Les points ELO sont automatiquement recalculés selon les nouvelles valeurs.

Supprimer un match

Cliquez sur l'icône poubelle, confirmez. Les ELO des deux joueurs concernés sont recalculés sans ce match.

Recherche et tri

  • Utilisez la barre de recherche pour filtrer par nom de joueur
  • Le menu Tri permet de classer par date croissante ou décroissante

7. Classement en direct

Accessible via la sidebar (Classement) ou depuis le tableau de bord. Le classement se met à jour automatiquement toutes les 10 secondes.

Projection en cours

Pendant une séance, cliquez sur Voir en plein écran depuis la page Séance pour afficher le classement en mode projecteur.

Badges du classement

  • Groupes de N : indique que les joueurs sont divisés en groupes (voir Règles)
  • Handicap actif : indique que le handicap de rang est activé (voir Règles)

8. Règles et défis

La page Règles et défis vous permet de personnaliser entièrement le système de points ELO et les mécaniques de jeu de votre cours.

Points ELO

Le tableau définit les points distribués selon l'écart de score entre les deux joueurs.

Écart de scorePoints GagnantPoints Perdant
0 à 6ConfigurableConfigurable
7 à 11ConfigurableConfigurable
12 à 20ConfigurableConfigurable

Modifiez les valeurs directement dans le tableau et cliquez sur Sauvegarder les points. Le bouton Réinitialiser restaure les valeurs par défaut.

💡

Les valeurs négatives (points perdant) représentent une perte d'ELO. Elles s'affichent en rouge dans le tableau.

Groupes de classement

Activez ce défi avec le toggle Groupes de classement. Quand il est actif :

  • Les joueurs sont divisés en groupes consécutifs selon leur rang (ex : groupe 1 = rangs 1 à 8)
  • Un joueur ne peut défier que les joueurs de son groupe
  • Vous choisissez la taille des groupes (nombre de joueurs par groupe)

Un aperçu des groupes s'affiche en temps réel quand vous ajustez la taille.

Handicap de rang

Activez ce défi avec le toggle Handicap de rang. Quand il est actif :

  • Si un joueur mieux classé défie quelqu'un de moins bien classé, il démarre le match avec un malus de points
  • Plus l'écart de rang est grand, plus le malus est élevé
  • Le tableau de handicap est entièrement configurable
✅

Ces deux mécaniques encouragent les joueurs à se défier entre pairs de même niveau, rendant les matchs plus équilibrés et motivants.

9. Mon compte

Cliquez sur votre nom en bas de la sidebar, ou accédez à Mon compte pour modifier vos informations personnelles.

Modifier le profil

Mettez à jour votre prénom, nom et email puis cliquez sur Enregistrer.

Changer le mot de passe

Saisissez votre mot de passe actuel, puis le nouveau mot de passe (à confirmer). Cliquez sur Changer le mot de passe.

Se déconnecter

Cliquez sur le bouton Déconnexion en bas de la page (ou depuis la sidebar). Une confirmation vous sera demandée.

10. Gérer les co-administrateurs

💡

Cette section est uniquement visible si vous êtes l'administrateur principal du cours. Un joueur administrateur temporairement n'y aura pas accès.

La page Administrateurs dans la sidebar liste tous les comptes ayant accès à l'espace admin.

Ajouter un co-administrateur

Un co-administrateur est un joueur promu depuis la page Joueurs. Éditez un joueur et activez l'option Nommer administrateur. Il aura alors accès à l'espace admin pour ce cours.

Retirer un co-administrateur

Sur la page Administrateurs, cliquez sur l'icône poubelle à droite de l'admin concerné, puis confirmez. Son accès admin est révoqué (son compte joueur reste intact).

⚠️

Vous ne pouvez pas supprimer votre propre compte administrateur depuis cette page.
